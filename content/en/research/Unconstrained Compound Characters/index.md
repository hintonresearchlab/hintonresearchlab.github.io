## About This Project

This project focuses on the development of intelligent systems for **Unconstrained Compound Character Recognition** using deep learning and computer vision techniques. The primary objective is to accurately recognize handwritten compound characters from unconstrained document images, particularly for Assamese and other Indic scripts.

The project addresses challenges such as:
- Complex handwritten character structures
- Variations in writing styles
- Noise and document degradation
- Character overlapping and segmentation difficulties
- Low-resource language datasets

The research integrates:
- Deep Learning
- Optical Character Recognition (OCR)
- Vision Transformers
- Sequence Modeling
- Attention Mechanisms
- State Space Models (Mamba)

to build robust handwritten script recognition systems.

## Research Objectives

The major objectives include:
- Recognition of unconstrained handwritten compound characters
- Robust feature extraction from document images
- Lightweight and efficient OCR architectures
- Handling complex ligatures and script variations
- Development of annotated handwritten datasets
- Improving recognition accuracy under real-world conditions

## Methodology

The proposed framework includes:
1. Handwritten document image collection
2. Image preprocessing and noise removal
3. Character segmentation and normalization
4. Deep feature extraction using CNN and Transformer architectures
5. Sequence modeling using attention and Mamba-based networks
6. Compound character classification and recognition

The project investigates architectures such as:
- CNN-LSTM Hybrid Models
- Vision Transformers (ViTs)
- CRNN-based OCR systems
- Attention-based recognition models
- Mamba and State Space Models (SSM)

## Applications

Potential applications include:
- Digital document preservation
- Historical manuscript digitization
- Assamese handwritten OCR systems
- Intelligent archival systems
- Educational and linguistic tools
- Regional language computing

## Results

The project aims to:
- Improve recognition accuracy for compound characters
- Build robust handwritten datasets
- Develop scalable OCR pipelines
- Enhance recognition performance under unconstrained settings

## Future Scope

Future directions include:
- Multilingual Indic OCR systems
- Real-time handwritten recognition
- Foundation models for document intelligence
- Large-scale Assamese document digitization
- Mobile-based handwritten text recognition systems
